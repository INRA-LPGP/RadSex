void distrib ( Parameters &  parameters)

Main function called by the distrib command.

Compute the distribution of markers between two groups.
This function creates a parsing thread which reads a markers table file and stores markers into a queue, and a processing thread which reads batches of markers from the queue and compute metrics.
After all markers are processed, the function generates an tabulated output file with columns:
Group1 | Group2 | Markers | P | Signif Bias

Parameters
parametersParameters object storing the value of all RADSex parameters
