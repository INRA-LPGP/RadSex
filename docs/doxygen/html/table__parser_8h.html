Read markers from a markers depth table and store them in a queue.

Read a markers depth table generated with process(). Each marker is stored in a Marker instance which is pushed to a queue. The function waits if the queue gets bigger than MAX_QUEUE_SIZE. Batches of markers are obtained from the queue with get_batch().

Parameters
parametersA Parameter object which contains values of all parameters
popmapA Popmap object with information on the group of each individual
markers_queueQueue of Marker objects
queue_mutexMutex to lock / unlock the markers queue
parsing_endedA flag indicating that the parsing ended, set to true before exiting the function
no_seqIf true, marker sequences and ids won't be stored in Marker objects
no_groupIf true, group counts won't be computed
